description: 브라우저 서브 에이전트를 파견하여 UI 렌더링, 반응형, 다크모드 무결성을 시각적으로 검증합니다.
---

1. **개발 서버 실행**:
   - 터미널에서 `npm run dev`를 실행하여 로컬 서버를 가동합니다.

2. **브라우저 에이전트 파견**:
   - 브라우저 서브 에이전트를 `http://localhost:3000`으로 이동시킵니다.
   - 사용자가 명시한 특정 페이지나 컴포넌트로 네비게이션합니다.

3. **체크리스트 검증**:
   - 반응형 테스트: 모바일 및 데스크톱 뷰포트 크기로 창을 조절하여 레이아웃 깨짐이 없는지 확인합니다.
   - 다크모드 테스트: 테마 토글 버튼을 클릭하여 다크 모드 전환 시 표(Table)나 텍스트 가독성이 유지되는지 확인합니다.
   - 인터랙션 테스트: 칸반 보드의 카드 클릭 등 주요 버튼이 정상 작동하는지 클릭 이벤트를 테스트합니다.

4. **결과 보고 (Artifact)**:
   - 브라우저 상호작용 과정을 녹화(Walkthrough Recording)하고 스크린샷을 캡처하여 아티팩트로 제출합니다.
   - 에러나 UI 깨짐이 발견되면 스스로 수정 패치를 제안합니다.
