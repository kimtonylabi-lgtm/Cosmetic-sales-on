description: 시니어 보안 리드 역할을 수행하여, 최신 코드 변경사항의 데이터 누출 및 보안 취약점을 감사(Audit)합니다.
---

1. **인증 및 접근 제어 검사**:
   - API 라우트 및 Server Actions에서 사용자의 세션(Session) 검증이 누락된 곳이 없는지 스캔합니다.
   - 비로그인 사용자가 `/dashboard` 등 보호된 라우트에 접근할 수 없는지 확인합니다.

2. **Supabase RLS(Row Level Security) 검토**:
   - 영업 담당자(REP)가 타인의 리드(Lead)나 딜(Deal) 데이터에 접근할 수 있는 쿼리가 있는지 분석합니다.
   - 클라이언트 컴포넌트에서 데이터베이스를 직접 호출하는 안티패턴이 있는지 스캔합니다.

3. **코드 레벨 취약점 검사**:
   - SQL 인젝션, XSS 등의 OWASP 주요 취약점이 있는지 확인합니다.
   - `.env` 파일에 저장되어야 할 API 키가 하드코딩된 곳이 있는지 점검합니다.

4. **수정안(Fix) 적용**:
   - 취약점이 발견되면 즉시 보고서를 작성하고, 안전한 코드로 교체하는 리팩터링을 수행합니다.
